> 前言：数据结构非常重要，下至底层操作系统，上至开发与复杂算法，实际上都是以计算机的角度将实体抽象为数据，再对数据进行组织，选择最合适的数据表示方法，然后运算。

> 数据结构中的指针概念：指针不仅是C++中独有的概念，实际上，表示对象的（位置的）量都可以视为指针。理解了指针的概念，我们可以更加轻松地学习数据结构。
